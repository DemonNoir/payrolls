# แผนปรับปรุงระบบวันหยุด/ลางาน (Leave System Overhaul)

## สรุปภาพรวม

ปรับระบบจากเดิมที่แยก 3 ปุ่ม (ทำ OT / ใช้วันหยุดสะสม / ลางาน) ➜ เหลือ **2 ปุ่ม**:

| เดิม (3 ปุ่ม) | ใหม่ (2 ปุ่ม) |
|---|---|
| ⏱ ทำ OT | ⏱ ทำ OT (เหมือนเดิม) |
| 📅 ใช้วันหยุดสะสม | 📅 ใช้วันหยุด/ลางาน ← **รวมทุกประเภท** |
| 🏖️ ลางาน ❌ (ยุบ) | — |

เมื่อเลือก "ใช้วันหยุด/ลางาน" จะมี Dropdown ให้เลือกประเภท + กรอกจำนวน **ชั่วโมง** (ไม่ใช่วัน)

## Auto-Accrual: ลาพักร้อน +4 ชม./เดือน

สูตร:
```
เดือนที่ทำงาน = จำนวนเดือนนับจากวันเริ่มงาน (startDate) ถึงวันนี้
วันหยุดสะสม = bankAdj (ปรับด้วยมือ) + OT สะสม + (เดือนทำงาน × 4 ชม.) − ชม.ที่ใช้ไป
```

---

## Data Model

### ของเดิม (kind: 'use')
```js
{ kind: 'use', hours: 8 }
```

### ของใหม่ (kind: 'use' แต่มี leaveType)
```js
{ kind: 'use', leaveType: 'annual', hours: 8 }   // ลาพักร้อน 8 ชม.
{ kind: 'use', leaveType: 'sick', hours: 4 }      // ลาป่วย 4 ชม.
{ kind: 'use', leaveType: 'swap', hours: 8 }      // สลับหยุด 8 ชม.
// ... รวม 8 ประเภท
```

> ⚠️ Backward compatible: ถ้า `leaveType` ไม่มี (ข้อมูลเดิม) → ถือว่าเป็น "ใช้วันหยุดสะสม (ไม่ระบุ)" ไม่หักเบี้ยขยัน

## Open Questions ที่ต้องตอบก่อนลงมือ

> [!IMPORTANT]
> ### Q1: "สลับหยุด" หักจากวันหยุดสะสมไหม?
> ปกติ "สลับหยุด" คือย้ายวันหยุดไปวันอื่น (ไม่ได้ใช้สิทธิเพิ่ม) ผมตั้งค่าไว้ว่า **หักจากวันหยุดสะสม** เหมือนกัน ถูกต้องไหมครับ? หรือไม่ต้องหัก?

> [!IMPORTANT]
> ### Q2: ลาคลอด / ลาบวช / ลาฌาปนกิจ หักเงินเดือนไหม?
> ผมตั้งค่าไว้ว่า **ไม่หักเงินเดือน** (แต่หักเบี้ยขยัน) ตามกฎหมายแรงงาน ถูกต้องไหมครับ?

> [!IMPORTANT]
> ### Q3: Auto-Accrual เริ่มนับจาก "วันที่เริ่มงาน" ในตั้งค่า?
> ถ้า startDate = 1 ม.ค. 2569, วันนี้ = 15 ก.ค. 2569 → ทำงาน 6 เดือน → ลาพักร้อนสะสม = 6 × 4 = **24 ชม.**  
> ใช้สูตรนี้ถูกต้องไหมครับ? หรือมี cap (เพดาน) เช่น ไม่เกิน 48 ชม./ปี?

> [!IMPORTANT]
> ### Q4: ลากิจ "รับค่าจ้าง" ที่บริษัทคุณมี cap กี่วัน/ปีไหม?
> หรือไม่ต้องสนใจ cap เพราะเป็นระบบเล็กๆ คำนวณส่วนตัว?

## Verification Plan

### Automated Tests
```bash
open tests/calc-tests.html
```

### Manual Verification
- กดวันที่ → เลือก "ใช้วันหยุด/ลางาน" → เลือกทุกประเภท → ดู preview ถูกต้อง
- บันทึกลาพักร้อน → เช็คว่าเบี้ยขยัน **ไม่หัก**
- บันทึกลาป่วย → เช็คว่าเบี้ยขยัน **โดนหัก** แต่เงินเดือน **ไม่หัก**
- บันทึกลากิจ ไม่รับค่าจ้าง → เช็คว่าทั้งเบี้ยขยันและเงินเดือน **โดนหัก**
- เช็ค auto-accrual: ตั้ง startDate แล้วดูว่ายอดวันหยุดสะสมเพิ่มขึ้น 4 ชม./เดือน
