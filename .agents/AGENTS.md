# Project Rules

## UI Redesign & Workflow Standard
ทุกครั้งที่มีการร้องขอให้รีดีไซน์ UI (User Interface) ให้ทำตามข้อกำหนดต่อไปนี้เสมอ:
1. **World-Class Design**: ให้ออกแบบในสไตล์พรีเมียมระดับโลก โดยต้องมีองค์ประกอบของ Glassmorphism, Micro-animations, และ Gradients ที่สวยงาม
2. **Walkthrough Documentation**: เมื่อทำการออกแบบและแก้ไขโค้ดเสร็จแล้ว ให้สร้างไฟล์ Walkthrough เพื่อสรุปการเปลี่ยนแปลง (แคปเจอร์หน้าจอ/วิดีโอ) และบันทึกไฟล์นี้ลงในโฟลเดอร์ `docs/`
3. **Version Control**: เมื่อทำงานทั้งหมดเสร็จสิ้น (แก้ไขโค้ดและสร้างเอกสารครบถ้วน) ให้ทำการ `git add`, `git commit` และ `git push` งานขึ้นไปยัง GitHub เสมอ
