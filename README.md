# 💰 Payrolls (ระบบคำนวณเงินเดือนและ OT อัตโนมัติ)

ระบบคำนวณเงินเดือนและค่าล่วงเวลา (OT) แบบครบวงจรที่ถูกออกแบบมาเพื่อความง่าย รวดเร็ว และครอบคลุมกฎหมายแรงงานไทย ตัวระบบถูกเขียนขึ้นด้วย Vanilla JavaScript, HTML5 และ CSS3 ซึ่งหมายความว่าทำงานได้บนเบราว์เซอร์ทันทีโดยไม่ต้องพึ่งพาเซิร์ฟเวอร์หรือฐานข้อมูลภายนอก (Data ทั้งหมดถูกเก็บอย่างปลอดภัยบน LocalStorage ของเครื่องคุณ)

## ✨ ฟีเจอร์หลัก (Key Features)

- **📆 ปฏิทินจดบันทึก OT**: เพิ่มและจัดการล่วงเวลา (OT) ในแต่ละวันได้ง่ายๆ พร้อมแสดงผลบนหน้าปฏิทินที่ดูเป็นมิตร
- **⏱ ตัวคูณล่วงเวลาหลายระดับ**: รองรับตัวคูณ OT อัตโนมัติ เช่น `1.0x` (วันธรรมดา/รับเป็นเงิน), `1.5x`, และ `3.0x` (วันหยุด)
- **🏖 จัดการวันหยุด**: สามารถเพิ่มวันหยุดล่วงหน้าแบบเจาะจงวัน หรือกำหนดเป็นช่วงระยะเวลา (เช่น 13-15 เม.ย.) ได้อย่างง่ายดาย
- **📈 สรุปแดชบอร์ด (Dashboard)**: สรุปรายรับ, สวัสดิการ, ยอดเงินเดือน, ค่าล่วงเวลาสะสม, รวมถึงคาดการณ์ **"รายได้สุทธิ"** ให้อัตโนมัติในแต่ละรอบบิล
- **💼 รองรับการคำนวณเงินเดือนระหว่างเดือน (Prorated Salary)**: ตั้งค่า "วันที่เริ่มงาน" ระบบจะทำการปรับสัดส่วนเงินเดือนให้เหมาะสมกับวันที่เข้าทำงานจริง
- **🏥 คำนวณประกันสังคมอัตโนมัติ**: หัก 5% ของฐานเงินเดือนให้โดยอัตโนมัติ (อิงจากกฎหมาย สูงสุดไม่เกิน 750 บาท/เดือน)
- **⚙ สวัสดิการและ KPI**: เพิ่มการคำนวณเบี้ยขยัน, ค่าอาหาร, ค่าเดินทาง, ค่ากะดึก และโบนัส KPI แบบรายวัน/รายเดือน
- **📥 ส่งออกข้อมูล (Export/Import)**: รองรับการส่งออกข้อมูลเป็นไฟล์ JSON เพื่อสำรองข้อมูล หรือพิมพ์เป็นรายงาน PDF ได้ทันที

## 🚀 การติดตั้งและใช้งาน (Getting Started)

เนื่องจากโปรเจกต์นี้เป็น Client-side 100% คุณไม่จำเป็นต้องรัน `npm install` หรือตั้งค่า Database ใดๆ 

1. **โคลนโปรเจกต์**
   ```bash
   git clone https://github.com/DemonNoir/payrolls.git
   ```
2. **เปิดใช้งาน**
   เพียงแค่ดับเบิลคลิกไฟล์ `index.html` หรือเปิดผ่านเบราว์เซอร์ (Chrome, Safari, Edge) 
   หรือใช้ Live Server Extensions ใน VSCode ได้เลย

## 📂 โครงสร้างโฟลเดอร์ (Project Structure)

เพื่อความง่ายในการพัฒนา โค้ดถูกแบ่งออกเป็นส่วนๆ (Modular) ดังนี้:

```text
payrolls/
├── index.html           # โครงสร้างหน้าเว็บหลัก
├── css/
│   └── style.css        # จัดการความสวยงามของระบบ (UI/UX)
└── js/
    ├── app.js           # จุดเริ่มต้นของระบบ จัดการ Event Click และ Routing ของหน้า
    ├── ui.js            # จัดการการเรนเดอร์ UI ทั้ง ปฏิทิน แดชบอร์ด และหน้ารายเดือน
    ├── data.js          # จัดการข้อมูล LocalStorage (การบันทึก/แก้ไข/ลบ/ดึงข้อมูล)
    ├── calc.js          # หัวใจหลักของระบบ! รวบรวมสูตรคำนวณเงินเดือน OT และสวัสดิการ
    ├── settings.js      # จัดการฟอร์มตั้งค่า และ วันหยุด
    └── utils.js         # ฟังก์ชันช่วยเหลือ เช่น จัดการวันที่ แปลงสกุลเงิน
```

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend**: HTML5, Vanilla CSS (Custom Variables)
- **Logic**: Vanilla JavaScript (ES5/ES6 compatible)
- **Storage**: Browser LocalStorage API
- **Charts**: HTML5 Canvas (Custom implementation)
