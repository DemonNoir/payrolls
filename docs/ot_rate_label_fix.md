# Walkthrough: แก้ไข Label และ Logic การแสดงอัตราค่าแรง OT

## สรุปการเปลี่ยนแปลง

### ปัญหาที่แก้ไข
1. **Dashboard:** การ์ด "อัตราค่าแรง OT" แสดง 54.17 บาท/ชม. ซึ่งจริงๆ คือ "อัตราค่าจ้างฐาน/ชม." ไม่ใช่ OT rate ที่ใช้จริง
2. **Entry Modal:** ข้อความ "อัตรา XX บาท/ชม." ไม่เปลี่ยนตามตัวคูณที่เลือก (1.0x/1.5x/3.0x) — แสดงฐานอย่างเดียวทุกกรณี

### สิ่งที่แก้ไข

#### A) Dashboard — แก้ Label + เพิ่มการ์ด OT เฉลี่ย
- เปลี่ยน label จาก **"⚡ อัตราค่าแรง OT (รอบปัจจุบัน)"** เป็น **"💼 อัตราค่าจ้างฐาน/ชม."**
- เพิ่มการ์ดใหม่ **"⚡ อัตรา OT เฉลี่ยรอบนี้"** = ยอดเงิน OT รวม ÷ ชม. OT รวม

#### B) Entry Modal — แสดง Effective Rate ตามตัวคูณ
- คำนวณ `effective_rate = base_hourly_rate × multiplier` แบบ real-time
- ตัวอย่าง (ฐาน 55.00 บาท/ชม.):
  - 1.0x → **"อัตรา 55.00 บาท/ชม."**
  - 1.5x → **"อัตรา 82.50 บาท/ชม."**
  - 3.0x → **"อัตรา 165.00 บาท/ชม."**

#### C) อื่นๆ
- แก้ label ด้านล่างปฏิทิน จาก "อัตราค่าแรง OT" เป็น "ค่าจ้างฐาน"
- แก้ label ด้านบนหน้าปฏิทิน (sub-header) จาก "ค่าแรง" เป็น "ค่าจ้างฐาน"
