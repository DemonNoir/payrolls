# PWA Icon Update Walkthrough

## สรุปการเปลี่ยนแปลง
- เพิ่มโฟลเดอร์ `icons/` สำหรับเก็บรูปภาพไอคอนของ PWA (Progressive Web App)
- ดาวน์โหลดและปรับขนาดรูปภาพไอคอนที่ผู้ใช้กำหนดเป็นขนาด **192x192** และ **512x512** พิกเซล
- บันทึกไฟล์รูปภาพเป็น `icon-192.png` และ `icon-512.png` ลงในโฟลเดอร์ `icons/`

## สิ่งที่ได้รับการทดสอบและยืนยัน
- ขนาดของไฟล์รูปภาพมีความถูกต้องตามที่ระบุในไฟล์ `manifest.json`
- โฟลเดอร์ถูกจัดเก็บไว้ในโครงสร้างโปรเจกต์อย่างถูกต้อง

*(คุณสามารถแคปหน้าจอตอนติดตั้ง PWA หรือบนหน้า Home Screen มาแปะที่นี่เพื่อยืนยันการแสดงผลไอคอนใหม่ได้)*
