# Walkthrough: สัญลักษณ์บ่งบอกกะดึกในปฏิทิน (Night Shift Indicator)

## 📌 สรุปการเปลี่ยนแปลง

เพิ่มสัญลักษณ์เพื่อแจ้งเตือนผู้ใช้งานเมื่อมีการเปิดตั้งค่า **"สัปดาห์นี้เข้ากะดึก"** เพื่อให้ทราบทันทีเมื่อดูที่หน้าจอ **ปฏิทิน** 

### สิ่งที่เพิ่มเติม
- **สัญลักษณ์ `🌙 (กะดึก)`:** จะแสดงต่อท้ายชื่อรอบบิล (เช่น "มกราคม 2026 🌙 (กะดึก)") ในหน้าปฏิทิน หากผู้ใช้มีการติ๊กเปิดโหมด "สัปดาห์นี้เข้ากะดึก" ในหน้าการตั้งค่า
- การแสดงผลจะเชื่อมโยงกับ Global Setting `nightEnabled` เพื่อให้ผู้ใช้สามารถรับรู้สถานะการคำนวณเงินกะดึกได้ทันทีโดยไม่ต้องกดกลับไปดูที่หน้าตั้งค่า

### การอัปเดตเวอร์ชัน
- อัปเดตแอปพลิเคชันจากเวอร์ชัน 24 เป็น **v=25** ในไฟล์ `index.html` และอัปเดตแคชใน Service Worker (`sw.js`) เป็น **ot-v13** เพื่อบังคับให้โหลด UI ใหม่ทันที (ตามกฎ Network-First strategy)

## 📁 ไฟล์ที่เกี่ยวข้องกับการแก้ไข
1. `js/ui.js` - แก้ไขฟังก์ชัน `renderCalendar()` เพื่อดึงสถานะจาก `settings()` และต่อท้ายข้อความลงใน `$('monthLabel').innerText`
2. `index.html` - บัมป์เวอร์ชันสคริปต์และสไตล์เป็น `?v=25`
3. `sw.js` - เปลี่ยนชื่อแคชเป็น `ot-v13` เพื่อเคลียร์แคชเก่าของผู้ใช้
