# Walkthrough: สัญลักษณ์บ่งบอกกะดึกในปฏิทิน (Night Shift Indicator)

## 📌 สรุปการเปลี่ยนแปลง

เปลี่ยนระบบกะดึกจากเดิมที่เป็น "การเหมาจ่ายทั้งรอบบิล" ให้สามารถ **เลือกระบุการเข้ากะดึกเป็นรายวัน** ผ่านปฏิทินได้โดยตรง เพื่อให้สามารถคำนวณค่ากะดึกได้อย่างแม่นยำตามวันที่เข้าทำงานกะดึกจริงๆ (ตอบโจทย์คนที่เข้ากะเป็นแบบสัปดาห์เว้นสัปดาห์)

### สิ่งที่เพิ่มเติม
- **ฟอร์มกรอกข้อมูลกะดึก:** เพิ่ม Checkbox `🌙 เข้ากะดึก (ได้รับค่ากะดึก)` ลงใน Modal สำหรับกรอกข้อมูลรายวัน ผู้ใช้สามารถติ๊กเลือกได้แม้ในวันที่ **ไม่มีการทำ OT** (ลงเป็น 0 ชั่วโมง) เพื่อให้ระบบนับเป็นวันเข้ากะดึกได้
- **สัญลักษณ์ `🌙` ในปฏิทิน:** จะมีไอคอนพระจันทร์เสี้ยวปรากฏที่มุมของวันที่ที่ผู้ใช้เลือกเข้ากะดึก ทำให้สามารถตรวจสอบได้ง่ายๆ จากหน้าปฏิทินรวม
- **ถอด Global Settings:** นำปุ่ม "สัปดาห์นี้เข้ากะดึก" แบบเหมารวมออกจากหน้าจอตั้งค่าหลัก เพื่อป้องกันความสับสนและให้การคำนวณเป็นไปในรูปแบบรายวันแทน

### การอัปเดตเวอร์ชัน
- อัปเดตแอปพลิเคชันจากเวอร์ชัน 25 เป็น **v=26** ในไฟล์ `index.html` และอัปเดตแคชใน Service Worker (`sw.js`) เป็น **ot-v14** เพื่อบังคับให้โหลด UI ใหม่ทันที 

## 📁 ไฟล์ที่เกี่ยวข้องกับการแก้ไข
1. `index.html` - เพิ่ม Checkbox กะดึกลง Modal, ลบจาก Settings และบัมป์ `?v=26`
2. `js/settings.js` - แก้ไขฟังก์ชัน `saveEntry` และ `openEntry` ให้บันทึกสถานะ `isNight` ต่อวันได้
3. `js/calc.js` - ปรับการคำนวณ `nightShiftDays` ด้วยลูปรายวัน แทนการใช้สวิตช์เดียวทั้งรอบบิล
4. `js/ui.js` - แสดงเครื่องหมาย `🌙` ในช่องวันที่บนปฏิทิน
5. `js/data.js` - นำคีย์ `night_shift_enabled` ออกจากการ Export/Import
6. `sw.js` - เปลี่ยนชื่อแคชเป็น `ot-v14`
