# Project Handoff

## Project Overview
โปรเจกต์ปฏิทิน OT (Overtime Calendar) เป็นแอปพลิเคชันบนเว็บสำหรับให้พนักงานบันทึกเวลาทำงานล่วงเวลา (OT), ลางาน, เข้ากะดึก, และวันหยุดนักขัตฤกษ์ โดยระบบจะคำนวณเงินเดือน, สวัสดิการ (ค่าเดินทาง, ค่าอาหาร, ค่ากะดึก, ค่าอาหาร OT) ให้อัตโนมัติ รองรับการบันทึกแบบ Single-day และ Batch Edit (หลายวัน)

## Current State
- บันทึกโค้ดเวอร์ชันล่าสุดขึ้น GitHub เรียบร้อยแล้ว (`main` branch)
- ใช้งานได้เสถียรทั้งบน Single-day modal และ Batch edit modal
- ปรับปรุงการคำนวณวันหยุดนักขัตฤกษ์และค่าอาหาร OT ให้ตรงตาม Business Logic ของบริษัทแล้ว

## Recent Changes (เซสชันล่าสุด)
1. **Multi-Rate OT (2 อัตราในวันเดียว):** 
   - รองรับการกรอก OT 2 อัตราในวันเดียวกัน เช่น 1x และ 3x (`js/settings.js`, `js/ui.js`)
   - แก้ไขโครงสร้างข้อมูลจาก `{ hours, multiplier, ... }` เป็น `{ rates: [{hours, multiplier}, ...] }`
2. **Auto-Holiday Logic:**
   - เมื่อเลือกตัวคูณ `1.0x` หรือ `3.0x` (ทั้งโหมด Single และ Batch) จะตั้งวันนั้นเป็น "วันหยุดนักขัตฤกษ์" ให้อัตโนมัติ
3. **Batch Edit Holiday Fix:**
   - แก้ไขบั๊กปุ่ม "บันทึกทับทั้งหมด" พังตอนโหลดหน้าจอ (ลบโค้ด listener เก่าใน `initMultiSelect` ใน `js/ui.js`)
   - อัปเดตให้ Batch Edit โหมดตั้งเป็นวันหยุดนักขัตฤกษ์ สามารถบันทึกเครื่องหมาย **เข้ากะดึก** พร้อมกันได้โดยข้อมูลไม่หาย
4. **OT Food Allowance Logic:**
   - ปรับเงื่อนไขใน `js/calc.js`: ถ้าเป็นวันทำงานปกติ ต้องทำ OT >= 2 ชั่วโมงถึงจะได้ค่าอาหาร OT
   - ถ้าเป็นวันหยุดนักขัตฤกษ์/วันอาทิตย์ ต้องทำงานรวม >= 10 ชั่วโมงถึงจะได้ค่าอาหาร OT
   - เพิ่มเอกสารรวบรวม Business Logic ไว้ที่ `docs/allowance_rules.md`
5. **Cache Busting:**
   - อัปเดต `index.html` (script `?v=31`) และ `sw.js` (เวอร์ชัน `ot-v19`) เพื่อบังคับล้างแคชให้ผู้ใช้ทันที
