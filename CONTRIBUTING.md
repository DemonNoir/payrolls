# คู่มือการมีส่วนร่วม (Contributing Guidelines)

ขอบคุณที่ให้ความสนใจที่จะเข้ามาร่วมพัฒนาโปรเจกต์ Payrolls (ระบบคำนวณเงินเดือนและ OT) ของเรา! 
โปรเจกต์นี้เปิดกว้าง (Open Source) สำหรับนักพัฒนาทุกคนที่ต้องการเข้ามาต่อยอดหรือปรับปรุงระบบให้ดียิ่งขึ้น

## กติกาเบื้องต้น (Code of Conduct)
- กรุณาใช้ถ้อยคำที่สุภาพและให้เกียรติซึ่งกันและกันในการพูดคุย หรือส่ง Issue
- พยายามอธิบายปัญหาหรือฟีเจอร์ที่ต้องการเพิ่มให้ชัดเจน

## วิธีการส่งโค้ดเข้ามาร่วมพัฒนา (Pull Request Process)

1. **Fork โปรเจกต์** ไปยังบัญชี GitHub ของคุณ
2. **สร้าง Branch ใหม่** สำหรับฟีเจอร์ หรือการแก้ไขของคุณ (ใช้ชื่อ branch ที่สื่อความหมาย)
   ```bash
   git checkout -b feature/awesome-new-feature
   # หรือ
   git checkout -b fix/calculate-bug
   ```
3. **ทำการแก้ไขโค้ด** 
   - กรุณาเขียนโค้ดให้เป็นระเบียบ (Clean Code) 
   - หากมีการแก้ไขส่วนของการคำนวณ (ใน `js/calc.js`) กรุณาทดสอบให้มั่นใจว่าไม่กระทบกับตัวแปรอื่น
4. **Commit ข้อมูลที่แก้ไข** (กรุณาเขียน Commit Message ให้ชัดเจนว่าแก้ไขอะไร)
   ```bash
   git commit -m "Add new holiday type calculation"
   ```
5. **Push โค้ดไปยัง Branch ของคุณ**
   ```bash
   git push origin feature/awesome-new-feature
   ```
6. **สร้าง Pull Request (PR)** บนหน้า GitHub 
   - อธิบายว่า PR นี้ทำอะไร แก้ไขปัญหาไหน (ถ้ามี Issue number ให้แนบมาด้วย)

## โครงสร้างและการแก้ไขโค้ดเบื้องต้น
หากคุณต้องการเพิ่มหน้า UI ใหม่ ให้ไปที่ `js/ui.js`
หากต้องการเพิ่มสูตรคำนวณสวัสดิการใหม่ ให้ไปที่ `js/calc.js` (และอย่าลืมเพิ่มใน `settings.js` หากต้องมีการกรอกค่า)

ขอบคุณที่เข้ามามีส่วนร่วมในการพัฒนาระบบนี้ให้สมบูรณ์แบบยิ่งขึ้นครับ! 🚀
